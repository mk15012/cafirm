CA Firm Pro App Icon

Matches the web app branding

How to use:

  1. Click "Download icon.png" button above
  2. Save the file to mobile/assets/icon.png
  3. Click "Download adaptive-icon.png" and save to same folder
  4. Commit and rebuild:
    git add -A && git commit -m "Add app icon"
    git push && cd mobile && eas build --platform android --profile preview
